基于GEC6818的餐厅自助点餐机的设计与实现
实现内容：
(1)餐厅自助机的UI设计：设计简洁、易用、符合餐饮场景的自助机界面，包括开机界面、登录界面、菜品浏览界面、下单确认界面、支付界面等，确保界面布局合理、操作逻辑清晰，适配GEC6818开发板的显示规格。
(2)font字库的实现：完成字体库的选型、移植与优化，确保自助机界面文字显示清晰、美观，支持中文、数字、常用符号等的正常展示，适配不同界面元素的字体大小与样式需求。
(3)自助机里面子菜单菜品的设计：设计菜品分类子菜单系统，可按菜品来划分菜单，每个菜品由名称、价格、图片等组成，提供菜品的快速浏览功能。
(4)每一个菜的mp4的播放：研发菜品播放视频模块功能，允许顾客点击该菜品显示相应的MP4格式介绍性视频，支持视频的播放、停止、退出等，实现视频播放与界面的兼容性。
(5)设计有餐厅特色的开机Logo和主题形象：设计有餐厅特色的开机Logo和主题形象，实现开机时Logo的展示，构建自助机的品牌知名度、视觉感。
(6)网络服务器接收顾客的订单：开发网络通信程序和服务端程序，通过基于网络的程序编写技术，建立自动机客户端与网络服务器的连接，顾客订单后订单信息即时上传至网络服务器，订单后台订单信息(菜品名称、数量、订单时间等)实时显示。订单状态的实时修改。
(7)制作一个app：开发配套的APP，实现餐厅管理功能，如菜品管理展示、订单管理与提醒顾客功能，确保APP与自助机、后台服务器的数据同步。
(8)点餐成功后通过蜂鸣器和LED提醒，按钮用来进行催单。
环境配置：
第一个文件夹内代码是APP，使用flutter实现，因此，请在vscode中配置flutter环境。
第二个文件夹内代码是上位机，使用QT5.7.0实现，因此，请在ubuntu中配置QT5.7.0交叉编译工具链并确保你的开发板可以运行QT。
我的QT环境：ubuntu22.04+QT5.7.0+GEC6818
